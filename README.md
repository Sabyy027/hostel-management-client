## Project Overview
The client application is built with React and Vite, providing a user interface for managing hostel operations.

## Technologies Used
- **React**: UI library
- **Vite**: Build tool
- **Tailwind CSS**: Styling
- **React Router DOM**: Routing
- **Axios**: HTTP client
- **Formik & Yup**: Form handling and validation
- **Lucide React**: Icons
- **Recharts**: Charts and visualizations
